# 花叔

我不会写代码。一行都不会。

但我用 AI 做出了 AppStore 付费榜 Top 1 的产品（小猫补光灯），写了 9 本技术书，造了一个能「蒸馏任何人思维方式」的开源框架。

所有产品，全部 AI 写的。我只负责想清楚要做什么。

### 我在做的事

**女娲.skill** — 把人的思维框架变成可运行的代码

不是让 AI 模仿某个人说话，而是提炼他们 *怎么想问题*。芒格的跨学科思考、费曼的第一性原理教学、塔勒布的反脆弱直觉——这些思维方式可以被结构化、被复用、被组合。我做了 21 个人物 skill，全部开源。

### 我相信的几件事

**思维框架比内容输出重要。** AI 可以无限生成内容，但决定内容质量的是背后的思考方式。这就是为什么我花大量时间做 perspective skills——不是为了让 AI 说漂亮话，而是为了让它 *想清楚*。

**实践是唯一的检验标准。** 我不写「AI 可能会怎样」的文章，只写「我用 AI 做了什么」的记录。每一篇教程、每一本橙皮书，都是真实项目的副产品。

**开源是最好的传播。** 好东西藏着没意义。女娲.skill 开源一周拿到 6000 star，不是因为我推广做得好，是因为大家真的需要它。
